# Gioco degli scacchi

Il programma deve:
- consentire a due giocatori di effettuare a turno mosse dei pezzi 
sulla scacchiera controllando che siano legali
- eliminare dal gioco i pezzi catturati dall'avversario


# AVVIO

Per eseguire il progetto seguire questi passaggi:

1. Creare un ambiente virtuale:
   python -m venv .venv

2. Attivare lâ€™ambiente virtuale:

   * Windows: .venv\Scripts\activate.bat
   * macOS / Linux: source .venv/bin/activate

3. Installare i moduli necessari:
   pip install -r requirements.txt

4. Solo per macOS installare Tkinter:
   brew install python-tk

5. Avviare il gioco:
   python main.py
